# Email Campaign Engagement Prediction Artifact
Predictive modelling workflow for email campaign open and click prediction using Logistic Regression and tree-based models.

This repository contains the code and presentation slides for the thesis artifact on predicting email campaign engagement for Online Marketing Group.
The artifact consists of a predictive analytics workflow for estimating email open and click engagement at the recipient-campaign level. The workflow uses recipient characteristics, campaign/content features, and historical engagement behaviour to train and evaluate predictive models.

## Repository contents
- Slides: presentation slides during thesis defense/company meeting
- Notebooks: staged analysis notebooks documenting the modelling workflow, including data preparation, feature engineering, model screening, hyperparameter tuning, final test evaluation, and model interpretation

## Data availability
The raw company data are not included in this repository due to confidentiality. As a result, the notebooks may not run end-to-end without access to the original data files. The code is provided to document the analytical workflow, modelling decisions, and evaluation procedure used in the thesis project.

## Main workflow

1. Data preparation and recipient-campaign dataset construction
2. Feature engineering and preprocessing
3. Model screening and hyperparameter tuning
4. Final model refitting and test-set evaluation
5. Model interpretation
